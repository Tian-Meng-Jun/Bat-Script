# Bat-Script

一些自己用的Bat脚本


# 1.Fuck QQ Update

> **删除QQ的自动下载的新版本更新，并保留当前版本文件

注意！在使用此脚本前，请在脚本内编辑QQ要保留的版本文件夹名。

建议在QQ更新前启用该脚本，防止QQ更新删除旧版本的文件。


# 2.QQ机器人服务定时重启脚本(战地五ROBOT版) _Beta版本

> **战地五ROBOT程序 定制版，但是Beta测试

在正式版进行功能更新和BUG修复前，一个用于测试稳定性的版本。

给战地五ROBOT程序定制的一个间隔一段时间自动重启QQ和ROBOT程序的脚本。

在使用此脚本前，请在脚本内编辑各项配置。

此脚本不保证能完全避免QQ被封号，只是可能减少QQ被封号概率而已。


# 3.QQ机器人服务定时重启脚本(战地五ROBOT版)

> **战地五ROBOT程序 定制版

给战地五ROBOT程序定制的一个间隔一段时间自动重启QQ和ROBOT程序的脚本。

在使用此脚本前，请在脚本内编辑各项配置。

此脚本不保证能完全避免QQ被封号，只是可能延长QQ被封号的时间和概率而已。


# 4.QQ机器人服务定时重启脚本

> **间隔一段时间重启QQ

间隔一段时间重启QQ，来减少因使用机器人插件导致QQ被封号的概率，但不保证能完全避免QQ被封号。


# 5.浏览器定时启动和关闭脚本

> **间隔一段时间启动和关闭浏览器

用于定时启用浏览器，运行某些内容，一段时间后关闭浏览器。

例如：晚签打卡，定时启用浏览器，运行签到脚本等。

当前脚本是按照运行签到脚本来写的，其他用途自行参考编辑。
